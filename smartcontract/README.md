# StacksGuard Smart Contracts

This directory contains the Clarity smart contracts for the StacksGuard investor protection layer.

## Contracts
- `refund-handler.clar`: Manages escrowed funds and trustless refund logic for failed projects.

## Development
- **Framework**: Clarinet
- **Language**: Clarity 4

## Commands
```bash
# Check syntax
clarinet check

# Run tests
npm run test
```
